# Medical Patient System

##  Description
This project is a Java-based system designed to manage patient records efficiently. It allows storing, updating, and viewing patient information.

##  Features
- Add new patients
- View patient records
- Update patient information
- Manage patient data efficiently

##  Technologies Used
- Java
- Eclipse IDE

##  How to Run
1. Open the project in Eclipse or IntelliJ
2. Run the main Java file
